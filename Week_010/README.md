## 学习方法
三分看视频理解、七分练习
- 视频：
  - 最佳办法：1.5～2.0倍数播放，难点（暂停 + 反复）
  - 最差办法：类似看美剧一样，原速看完，且仅看一遍
  
**摒弃"旧习惯"-最重要**
- 不要死磕(传统方式)，要用无毒神掌，要敢于放手，敢于死记硬背代码，要勤看高手代码，尤其是国际版的高票代码
- 最佳方式：5分钟想不出来，直接看题解或高票代码，用五毒神掌变成自己的东西。
- 最差方式：自己单挑，不借助外部帮自己解决，以为15-30分钟可以搞定，谁知道死磕了2-3小时或者一晚上，终于"通过"。
